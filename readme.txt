Auction
--------------

Auction example made using multiagent system implemented in Java usng Jade.
The type of auction is an English auction where the biggest bid is winner.


Instructions for runnig auction in console:
-download netbeans (https://netbeans.org/downloads/6.1/index.html)
-download Jade (https://jade.tilab.com/)
-open java project in netbeans
-build project
-run project
-start 2 bidding agents
-start 1 auction agent and as argument add item name and item price (Picture1, 100)
-the auction results are displayed in the console
